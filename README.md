# App Lista de Filmes com React Native Expo

## Objetivo principal:  

Compreender o uso do React Navigation e REST API em React Native / Expo  

### Dependências Utilizadas

Este projeto utiliza as seguintes bibliotecas:

- @react-navigation/native
- @react-navigation/native-stack
- react-native-screens
- react-native-safe-area-context
- react-native-paper

As dependências já estão configuradas — basta rodar `npm install`.

## Instalação

```bash
git clone https://github.com/AmonAmarth2003/App-lista-de-filmes-com-React-Native-Expo-
cd App-lista-de-filmes-com-React-Native-Expo-
npm install
npx expo start
```
